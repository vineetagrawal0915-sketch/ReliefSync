# ReliefSync | Disaster Coordination Platform 📡🛡️

ReliefSync is a high-fidelity, full-stack coordination platform designed for real-time disaster response. Built with a premium "Glassmorphism" aesthetic, it empowers Government Admins, NGOs, and Volunteers to collaborate seamlessly during crises.

## 🚀 Key Features
- **Real-Time Tasking:** Deploy and track rescue mission status across affected zones.
- **Role-Based Security:** Custom interfaces for Admins (Mission Control), NGOs (Field Ops), and Volunteers.
- **Global Inventory:** Dynamic resource tracking for food, medicine, and personnel.
- **Operational Map:** Visual geographic tracking of active emergencies and responder distribution.
- **Premium Dark UI:** Modern, high-contrast interface designed for low-light command centers.

## 🛠️ Tech Stack
- **Frontend:** React, Tailwind CSS, Lucide Icons, Leaflet Maps
- **Backend:** Node.js, Express.js
- **Database:** Supabase (PostgreSQL)
- **Security:** JWT Auth & Role-Based Access Control (RBAC)
